# 採購單系統 2

## 這是甚麼？
- 這個系統是為了改善我爸的工作體驗而製作的系統。

## 平台需求
- 最低 Windows 7
- 建議 Tomcat 9.0
- 最低 MySQL 5.7
- 最低 Java 7

## 注意事項
- 使用Windows 7時，需要把Driver的路徑改為 **com.mysql.jdbc.Driver**
- 使用前，需要使用 **purchase_order_script.sql** 創建database
- 使用前，需要將 **purchase_order_2.xml** 放在 **.../Tomcat 9.0/conf/Catalina/localhost** 底下
